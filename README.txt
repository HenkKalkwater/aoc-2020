Advent of Code 2020
================================================================================

HOW TO COMPILE:
Run "dub".


HOW TO RUN:
./oac-2020 [DAY] [PART] <INPUT_FILE> <DAY_SPECIFIC_ARGUMENTS...>

The program reads it input from $PWD/in/<DAY>.txt if INPUT_FILE is not specified or from stdin 
if INPUT_FILE is equal to "-".
